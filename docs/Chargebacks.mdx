# Chargebacks

Chargebacks occur when a customer disputes a payment with their bank. Vatly notifies you via webhooks when chargebacks happen. {{ className: 'lead' }}

## The Chargeback Resource

Below you'll find all properties for the Vatly Chargeback resource.

### Properties

<Properties>
  <Property name="id" type="string">
    Unique identifier for the chargeback (`chb_...`).
  </Property>
  <Property name="status" type="string">
    The status: `open`, `won`, or `lost`.
  </Property>
  <Property name="orderId" type="string">
    The disputed order ID.
  </Property>
  <Property name="amount" type="integer">
    Chargeback amount in cents.
  </Property>
  <Property name="currency" type="string">
    Three-letter ISO currency code.
  </Property>
  <Property name="reason" type="string|null">
    Reason provided by the bank.
  </Property>
  <Property name="testmode" type="bool">
    Whether this is a test chargeback.
  </Property>
  <Property name="createdAt" type="string">
    Creation timestamp (ISO 8601).
  </Property>
</Properties>

## Retrieve a chargeback {{ tag: 'GET', label: '/v1/chargebacks/:id' }}

<Row>
  <Col>

    Retrieve a chargeback by its ID.

  </Col>
  <Col sticky>

    <CodeGroup title="Request" tag="GET" label="/v1/chargebacks/:id">

    ```php {{ title: 'PHP' }}
    $chargeback = $vatly->chargebacks->get('chb_abc123');

    echo $chargeback->status;
    echo $chargeback->amount / 100 . ' ' . $chargeback->currency;
    echo $chargeback->reason;
    ```

    </CodeGroup>

  </Col>
</Row>

## List all chargebacks {{ tag: 'GET', label: '/v1/chargebacks' }}

<Row>
  <Col>

    Retrieve a paginated list of all chargebacks.

    ### Optional attributes

    <Properties>
      <Property name="limit" type="integer">
        The number of chargebacks to return (default: 10, max: 100).
      </Property>
      <Property name="startingAfter" type="string">
        A cursor for pagination.
      </Property>
    </Properties>

  </Col>
  <Col sticky>

    <CodeGroup title="Request" tag="GET" label="/v1/chargebacks">

    ```php {{ title: 'PHP' }}
    $chargebacks = $vatly->chargebacks->list();

    foreach ($chargebacks as $chargeback) {
        echo $chargeback->id . ': ' . $chargeback->status;
    }
    ```

    </CodeGroup>

  </Col>
</Row>

---

## Chargeback statuses

| Status | Description |
|--------|-------------|
| `open` | Dispute is ongoing |
| `won` | You won the dispute |
| `lost` | You lost the dispute |
